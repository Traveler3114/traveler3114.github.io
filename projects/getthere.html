Overview

A modular transport platform with a unified wallet that integrates multiple mobility operators through a plugin architecture.

The Challenge

Travel and mobility apps are fragmented, where users need separate accounts, wallets, and tickets for different transport modes.

My Contribution

A modular transport platform with a unified wallet and account system across multiple mobility operators. Designed the backend architecture to allow rapid integration of new operator APIs via a plugin pattern. Implemented core wallet system and plugin based operator integration with a focus on clean, extensible architecture.

Technical Highlights

Designing a plugin system that lets new operators be added without modifying core code. Solved with interface based contracts and dynamic service registration.

Results

Provides a unified account and wallet system across multiple transport operators from a single platform.
