Pigeon Invaders

Duration: Multiple iterations Role: Sole Developer
C++C#PythonRaylibASP.NETMySQL

Overview

A game rebuilt across Python, C#, and C++ stacks to demonstrate cross-platform architectural skill and progressive performance improvement.

The Challenge

Porting a game across different languages and frameworks is the fastest way to surface architectural weaknesses, but most projects optimize for one stack and never build the cross platform insight that makes architecture truly portable.

My Contribution

A game rebuilt in three versions: Python/Pygame (rapid prototype), C#/Windows Forms + Firebase (desktop with cloud storage), and C++/Raylib + ASP.NET API + MySQL (high performance with full backend). Each iteration improved architecture and performance, demonstrating adaptability across stacks.

Technical Highlights

Porting game logic between fundamentally different frameworks while preserving gameplay feel and improving performance. Solved by abstracting core game rules and reimplementing rendering and input layers.

Results

Three complete, playable versions across Python, C#, and C++ stacks, each with progressively better architecture, performance, and backend integration.

Metrics: 3 complete rewrites · 3 distinct tech stacks
